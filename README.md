# Simple Story App

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/simple-story-app.git
```

### Step 2

Change the directory to simple-story-app

```shell
cd simple-story-app
```

### Step 3

Run project

```shell
python main.py
```
